На данном этапе выполнены первые две задачи из списка задач, необходимых для достижения поставленной цели.
А именно, изучены существующие протоколы отрицаемого шифровани и изучен алгоритм решения квадртаных уравнений в бинарном конечном поле. 
В рамках первой задачи были рассмотрены имеющиея протоколы, проведено сравнения протоколов с аналогами.
В качестве первого этапа работы был преложен протоколо отрицаемого шифрования с алгоритмом решения кубических уравнений в простом конечном поле,
на основе которого будет разрабатываться навый протокол и который предложен в качестве одного из аналогов.
Для реализации и проверки предлагаемого алгоритма выбран язык программирования и среда разработки, а также реализованы базовые методы, необходимые для реализация алгоритма.
