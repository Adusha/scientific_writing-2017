## Сравнение аналогов
### Обращение в автосервис напрямую
При обращении в автосервис напрямую, клиент часто получает неверную информацию об услугах и их цене. Клиент не знает о компании, о качестве услуг и о возможном обмане. 
### Мобильное приложение fixam
Отсутствие карты с компаниями, отсутствие возможности записаться в конкретную компанию, на конкретную услугу, заявленные цены не соответствуют реальным.
### Веб-приложение Uremont
Отсутствует клиент под смартфон, медленная работа карты, для создания заявки необходимо звонить в автосервис.

## Критерии для сравнения
* Запись онлайн - главная возможность, предоставляемая приложением: запись онлайн на нужную услугу;
* Рейтинг компаний - составляется из оценок пользователей. Необходим для повышения качества оказываемых услуг и удобства пользователей.
* Удобство использования - удобно ли использовать приложение для заказа услуг; 
* Карта компаний - наличие карты компаний для навигации прямо из машины клиента.
